CCUsageWidget 설명

동작하는 컴퓨터 + 등록된 ip 들이 ai 사용량을 ccusage cli 를 활용해 조회하여 실시간으로 나타내는 컴퓨터 바탕화면 위젯



위젯 세팅 기능

* [ ] ccusage 모듈 사용을 위한 필요 모듈 혹은 프로그램 설치(ex. node, npm, ccusage, ...)
* [ ] 등록된 ip ssh 테스트



위젯 ui 동작 기능

* [ ] "X" 버튼을 누르면 위젯 종료
* [ ] 최소화/최대화 버튼을 통해 위젯 최소화, 최대화
* [ ] 플러스 버튼을 누르면 ssh를 통해 데이터를 받아올 컴퓨터의 ip등록
* [ ] 좌/우 버튼을 누르면 등록된 ip들 전환



위젯 데이터 동작 기능

* [ ] 현재 ui에 보이도록 지정된 ip 에서 ccusage daily, ccusage monthly 데이터 조회
* [ ] 조회한 데이터를 세로로 "일일 조회" 영역과 "월 조회" 영역으로 나누어서 보여줌
* [ ] claude, codex, gemini 영역으로 가로로 나누어서 보여줌 (만약 특정 agent의 데이터가 없다면 "없음" 으로 표시)



위젯 추가 기능

* [ ] os 에 맞춰 현재 ip의 별칭 지정(윈도우, 우분투, 맥 등)
* [ ] ssh 연결 테스트해서 연결이 안되면 연결 안됨으로 데이터 영역 대체
